Key Features

Raycasting & Rendering

Accurate 3D rendering achieved through raycasting, allowing the player to experience realistic depth and perspective within the maze.

Orientation & Rotation

Real-time camera controls ensure smooth player navigation with full rotational capabilities for an immersive experience.

Wall Rendering & Collision Detection

Enhanced wall rendering with distinct colors based on orientation and collision detection to prevent players from moving through walls.

About the Project

"The Maze" is a portfolio project that I developed to demonstrate my skills in raycasting, game mechanics, and real-time rendering. The inspiration came from my interest in creating an immersive 3D environment that could be navigated in real-time.

Goals Achieved:

  • Raycasting and Rendering: Successfully implemented, allowing accurate 3D environment rendering.
  • Orientation and Rotation: Real-time camera controls are functioning smoothly, enhancing player navigation.
  • Wall Rendering: Improved wall rendering differentiates between NORTH/SOUTH and EAST/WEST walls.
  • Collision Detection: Accurate collision detection prevents the player from passing through walls.
  • Map Parsing: The map parser loads maze layouts from external files for easy modifications.
  • 2D Minimap: A toggleable 2D minimap assists with navigation.

Remaining Challenges:

  • Enemy Integration: Developing enemy behaviors and interactions with the player remains a challenge.

Adaptations and Next Steps:

  • Focus on Enemy Integration: Defining enemy behavior and AI.
  • Ongoing Enhancements: Refine features like texture integration and gameplay balance.
  • Documentation and Maintenance: Keep updating the project documentation and ensure code quality.
